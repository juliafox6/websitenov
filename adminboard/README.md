# Admin Board

Админ-панель для управления базой данных книжного магазина.

## Запуск

Для запуска админ-панели выполните:

```bash
cd adminboard
python admin.py
```

Админ-панель будет доступна по адресу: http://localhost:5001

## Вход

По умолчанию пароль администратора: `12345`

Пароль можно изменить в файле `admin.py`, изменив значение переменной `ADMIN_PASSWORD`.

## Функции

1. **Добавление книг** - форма для добавления новых книг в базу данных
2. **Поиск пользователей** - поиск пользователей по имени или фамилии
3. **Управление пользователями**:
   - Изменение тарифа пользователя
   - Просмотр и изменение статусов заказов
   - Оценка повреждений книг при возврате заказа
   - Автоматический расчет штрафов и возврат депозита

## Примечание

Админ-панель использует ту же базу данных, что и основное приложение (`instance/mainDB.db`).
